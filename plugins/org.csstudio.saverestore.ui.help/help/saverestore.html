Save, Set & Restore

Overview

Save, Set & Restore provides facilities to simultaneously monitor several PVs, store their values, compare the live or stored values with the values at other times, and restore the previously stored values. Application also provides mechanisms to browse and manage the stored sets as well as to export the values to standard file formats.

Getting Started

The two main entities of the Save, Set & Restore application (SSR) are Beamline Set and Snapshot. Beamline Set defines the list of PV names, which can be loaded into the spreadsheet, while the Snapshot is a recording of PV values defined by the Beamline Set at one specific time. Snapshot always belong to a particular Beamline Set, which is also the way how the entities are made available in the SSR Browser.

The user interface of SSR is composed of three different parts: Save & Restore Browser (view), Snapshot Viewer (Editor), and Beamline Set Editor (Editor). Browser provides mechanisms to browse and select Beamline Sets and Snapshots, while the two editors provide facilities, to create or display the contents of the SSR entities.

Browser

Depending on your storage provider the browser is divided into two to four different parts:

  • Branch Selector (Optional) is available in the drop down menu. It displays the list of available branches as provided by the storage provider. A branches can be anything that provides a group of beamline sets, snapshots (and base levels if supported).
  • Base Level Browser (Optional) is available as the top most part of the browser view. The base level defines a distinctive set of beamline sets, for example for one particular experiment
  • Beamline Set Browser provides a list of all available Beamline Sets for the selected branch and base level. If branches and base levels are not supported, the Beamline Set Browser is the top most selector that is available. The beamline sets are ordered hierarchically and are displayed in a tree-like view.
  • Snapshot Browser provides a list of all available Snapshots for the selected Beamline Set. The snapshots are order by time (latest on top) and are displayed together with the creation date, creator name and message/description.
